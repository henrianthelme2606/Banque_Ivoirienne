## Analyse des données bancaires d'une banque fictive - Côte d'Ivoire

##  Objectif
L'objectif de cette analyse etait d'aider la banque à améliorer ses octrois de crédit et de réduire les impayés
##  Données
- Source : Dataset_Banque_Ivoirienne_Credit.xls
- Variables : Ville, age_client, nombre_enfants, etc.

## Analyse réalisée
- Analyse descriptive
- Visualisation des données
- Étude des profils clients
- Corrélations entre variables

##  Modèle prédictif
Le modèle de régression logistique permet de prédire efficacement les risques d’impayés.

##  Résultats
Cette étude a permis d’identifier plusieurs facteurs influençant le défaut de paiement, notamment :
•	le faible revenu, 
•	le niveau élevé d’endettement, 
•	certains profils familiaux, 
•	et certaines zones géographiques. 

## Recommandations
La banque pourrait :
•	renforcer l’analyse du taux d’endettement, 
•	limiter certains montants de crédit, 
•	améliorer le scoring client, 
•	et mettre en place une surveillance des profils à risque.
